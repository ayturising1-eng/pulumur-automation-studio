# Pülümür Automation Studio V8.2.4

Taban: V8.2.3 çalışan paket.

Bu sürümde sadece hedefli düzeltmeler yapıldı:
- Ölçülendirme geçici olarak kapatıldı.
- Ayna yan görünüşte bloklar gerçek mirror block definition ile üretiliyor.
- Ayna görünüşte sağ yan görünüş dikme alt ucu, ön görünüş dikme alt ucu ile aynı -Y hattına hizalanıyor.
- Tablolardaki metinler hücre içine sığacak şekilde dinamik kırılıyor/küçülüyor.
- Tablo hücre metinleri DXF içinde düzenlenebilir MTEXT olarak yazılıyor.
- V8.2.3 geometri tabanı korunmuştur.
